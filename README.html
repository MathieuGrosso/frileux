Setup

Installation 
npm install

# Variables d'environnement
cp .env.example .env

# Lancer le serveur de dev
npx expo start
Supabase 
# 1. Crée un projet sur supabase.com
# 2. Exécute la migration initiale
supabase db push

# 3. Déploie la Edge Function (suggestion IA)
supabase functions deploy suggest-outfit

# 4. Ajoute la clé Anthropic comme secret
supabase secrets set ANTHROPIC_API_KEY=sk-ant-...

Variables requises

EXPO_PUBLIC_SUPABASE_URL EXPO_PUBLIC_SUPABASE_ANON_KEY EXPO_PUBLIC_OPENWEATHER_API_KEY ANTHROPIC_API_KEY

Structure

app/ — Expo Router (file-based routing)
├── _layout.tsx — Auth guard + providers
├── auth/ — Login / Register
├── (tabs)/
│ ├── index.tsx — Aujourd'hui (météo + suggestion + photo)
│ ├── history.tsx — Historique des tenues
│ └── circle.tsx — Cercle privé
├── outfit/[id].tsx — Détail d'une tenue
└── settings.tsx — Profil + niveau frilosité
 
components/ — Composants réutilisables
lib/ — Supabase client, météo API, types
supabase/
├── migrations/ — Schema PostgreSQL
└── functions/ — Edge Function: suggest-outfit
